Hallo, Daar

Dit is een boek over iets dat Dydra Studio heet. Het woont in je webbrowser, wat best handig is omdat je het dan niet ergens in een la hoeft te zoeken.

Dydra Studio helpt je om data bij te houden. Data is een woord voor dingen die je weet en wilt onthouden. De data woont in kleine verzamelingen die repositories heten, wat een nogal deftig woord is voor "een plek waar dingen bewaard worden." (Zoals een kast, maar dan voor informatie in plaats van jam.)

Wat kan het?

Dydra Studio is wat slimme mensen een "single-page web application" noemen, wat betekent dat als je eenmaal aangekomen bent, je niet steeds ergens anders heen hoeft. Alles gebeurt precies waar je bent, wat veel comfortabeler is.

Dit zijn de dingen die je kunt doen:

  • Je data bewaren in repositories (veilig en wel)
  • Vragen stellen over je data (en antwoorden krijgen, meestal)
  • Data binnenhalen van andere plekken (importeren)
  • Data meenemen om te delen met vrienden (exporteren)
  • Je favoriete vragen opslaan zodat je ze opnieuw kunt stellen (views)
  • De dingen veranderen die over jezelf geschreven staan
  • Delen met anderen (als je dat wilt)

Voordat we beginnen

De deur vinden

Om bij Dydra Studio te komen, moet je dit adres bezoeken:

Het Adres: https://dydra.com/ui/user

Typ dat in je browser, en daar ben je. Het is een beetje zoals het juiste pad naar iemands huis kennen.

  • De Gebruikers App: Dit is de vriendelijke deur voor gewone mensen. Daar gaan we het over hebben.

Snel aan de slag (voor wie haast heeft)
