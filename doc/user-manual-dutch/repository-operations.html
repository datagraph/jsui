Waarin we plekken vinden om dingen te bewaren

Een repository is een plek waar data woont. Zie het als een gezellig huisje voor je informatie‚Äîeen plek waar feiten en cijfers zich kunnen settelen en comfortabel kunnen zijn. Je kunt zoveel repositories hebben als je wilt, elk met zijn eigen naam en zijn eigen verzameling dingen erin.

Een nieuwe repository maken

Als je een nieuwe plek wilt om wat data te bewaren, zo maak je er een:

Stap 1: Ga naar je accountpagina. Daar staan al je repositories.
Stap 2: Zoek de Nieuwe Repository knop en klik erop.
Stap 3: Er verschijnt een klein venstertje dat om een naam vraagt. Bedenk een goede naam‚Äîiets dat je herinnert aan wat erin zit. Je kunt ook een type kiezen:
  • lmdb: Een gewone repository (dit is de gebruikelijke keuze)
  • lmdb-revisioned: Een speciale repository die onthoudt hoe dingen waren, als een dagboek

Figuur 1: Geef je nieuwe repository een naam.

Stap 4: Klik op OK (of druk op Enter).
Stap 5: En daar is hij! Je nieuwe repository, helemaal klaar voor data.
Over namen: Repository-namen kunnen letters, cijfers, koppeltekens (-) en underscores (_) hebben. Geen spaties echter. Spaties verwarren computers. Houd het simpel.

Wat zit er in een repository

Als je naar een repository kijkt, zie je:

  • De naam (die koos je zelf)
  • Een samenvatting (een paar woorden over wat erin zit)
  • Een beschrijving (meer woorden, indien nodig)
  • Een homepage (een webadres, als die er is)
  • Hoeveel quads erin zitten (quads zijn de kleine stukjes data)
  • Hoeveel ruimte het inneemt
  • Licentie-informatie

  • Views (vragen die je opgeslagen hebt)

Als er iets misgaat

Import werkt niet

  • Is het een ondersteund bestandstype?
  • Is het bestand geldig?
  • Werkt je internet?
  • Heb je toestemming om naar deze repository te schrijven?

Export werkt niet

  • Zit er eigenlijk wel data in?
  • Probeer een ander formaat
  • Check de download-instellingen van je browser
