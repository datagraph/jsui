Waarin we over onszelf leren

Je account is een verzameling dingen die zeggen wie je bent. Het is een beetje zoals een kaartje met je naam erop, behalve dat het meer informatie heeft dan dat, en het woont in de computer in plaats van in je zak.

Wat je daar vindt

Als je naar je account kijkt, zie je allerlei dingen over jezelf:

  • Je gebruikersnaam (de naam waarmee je binnenkomt)
  • Je e-mailadres (voor belangrijke berichten)
  • Je volledige naam (degene die je ouders je gaven)
  • Je homepage en blog (als je zulke dingen hebt)
  • Je bedrijf (waar je werkt)
  • Waar je woont
  • Je repositories (die komen later in het boek)

Je geheime token

Je hebt iets dat een "token" heet. Het is als een speciaal wachtwoord, maar voor computerprogramma's om te gebruiken in plaats van mensen. Als je programma's schrijft die met Dydra moeten praten, gebruiken ze deze token.

Je token vinden

  1. Ga naar je accountpagina
  2. Zoek het Token-gedeelte
  3. Daar is hij!

Een nieuwe token krijgen

Soms heb je een nieuwe nodig. Misschien heb je per ongeluk je oude aan iemand verteld (oeps), of misschien wil je gewoon een frisse start.

  1. Zoek het token-gedeelte
  2. Klik op Reset Token
  3. Een gloednieuwe token verschijnt!
  4. Als er programma's de oude gebruikten, moet je ze over de nieuwe vertellen

Figuur 3: De Reset knop. Gebruik hem verstandig.

Een waarschuwing: Je token is geheim, net als een wachtwoord. Vertel het aan niemand die je niet volledig vertrouwt. Als iemand anders je token heeft, kan hij doen alsof hij jou is. Als je denkt dat iemand het gezien heeft, reset het dan meteen.
