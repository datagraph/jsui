Appendix A: Supported File Formats

Import Formats

The following file formats are supported for importing data into repositories:

Format File Extensions Description
Turtle .ttl RDF serialization format
RDF/XML .rdf, .xml XML-based RDF format
N-Triples .nt Line-based RDF format
N-Quads .nq N-Triples with graph context
TriG .trig Turtle with named graphs
JSON-LD .jsonld, .json JSON-based RDF format
CSV .csv Comma-separated values

Export Formats

The following formats are available for exporting repository data:

Format File Extension Description
Turtle .ttl RDF serialization format
N-Triples .nt Line-based RDF format
N-Quads .nq N-Triples with graph context
RDF/XML .rdf XML-based RDF format
JSON-LD .jsonld JSON-based RDF format
TriG .trig Turtle with named graphs
CSV .csv Comma-separated values

Query Result Formats

When executing SPARQL queries, you can request results in the following formats:

Format Media Type Description
JSON application/sparql-results+json Structured JSON results (default)
XML application/sparql-results+xml XML format results
SVG image/vnd.dydra.SPARQL-RESULTS+GRAPHVIZ+SVG+XML Graph visualization
HTML text/html Formatted HTML table
CSV text/csv Comma-separated values
TSV text/tab-separated-values Tab-separated values
Turtle text/turtle RDF Turtle format
N-Triples application/n-triples N-Triples RDF format
RDF/XML application/rdf+xml RDF/XML format
JSON-LD application/ld+json JSON-LD format
SSE application/sparql-query+sse SPARQL query SSE format

Appendix B: Keyboard Shortcuts

The following keyboard shortcuts are available in Dydra Studio:

Action Shortcut
Execute SPARQL Query Ctrl+Enter (Windows/Linux) or Cmd+Enter (Mac)
Submit Form Enter
Cancel Dialog Escape
Close Tab Click X button on tab

Appendix C: Glossary

Key terms used in this manual:

Term Definition
Account A user account on the Dydra platform
Repository A container for RDF data
View A saved SPARQL query
Quad An RDF statement (subject, predicate, object, graph)
SPARQL Query language for RDF data
Collaborator An account with access to a repository
Token Authentication token for API access
Pane A tabbed interface element for viewing content
Import Adding data to a repository
Export Downloading data from a repository
