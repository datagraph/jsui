## Image Specifications

- **Format**: PNG
- **Width**: Approximately 1200px (will be scaled to 50% page width in the manual)
- **Content**: Browser screenshots showing the relevant interface elements
- **Timing**: Images should show the state after values are entered but before operation buttons are pressed (except for confirmation dialogs which show the dialog itself)

## Capturing Screenshots

When capturing screenshots:
1. Use a modern browser (Chrome, Firefox, Safari, or Edge)
2. Capture at a resolution that shows all relevant UI elements clearly
3. Ensure form fields show entered values
4. Highlight or indicate the button/action that will be performed next
5. For confirmation dialogs, show the full dialog with warning messages visible
