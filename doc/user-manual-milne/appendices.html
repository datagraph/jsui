Appendix A: File Types

Computers like to know what kind of file they're looking at. Here are all the kinds that Dydra understands.

For Bringing Data In

Type File Ending What It Is
Turtle.ttlA friendly way to write RDF
RDF/XML.rdf, .xmlRDF in XML format (lots of angle brackets)
N-Triples.ntOne fact per line, very simple
N-Quads.nqLike N-Triples, but with an extra piece
TriG.trigTurtle with named graphs
JSON-LD.jsonld, .jsonRDF that looks like JSON
CSV.csvLike a spreadsheet (commas between things)

For Taking Data Out

Type File Ending What It Is
Turtle.ttlA friendly way to write RDF
N-Triples.ntOne fact per line
N-Quads.nqOne fact per line, with graph names
RDF/XML.rdfRDF in XML format
JSON-LD.jsonldRDF that looks like JSON
TriG.trigTurtle with named graphs
CSV.csvCommas between everything

For Query Answers

Type Technical Name What It Is
JSONapplication/sparql-results+jsonThe usual answer format
XMLapplication/sparql-results+xmlFor people who like XML
SVGimage/vnd.dydra.SPARQL-RESULTS+GRAPHVIZ+SVG+XMLA picture!
HTMLtext/htmlA nice table you can read
CSVtext/csvGood for spreadsheets
TSVtext/tab-separated-valuesAlso good for spreadsheets
Turtletext/turtleFor CONSTRUCT queries
N-Triplesapplication/n-triplesFor CONSTRUCT queries
RDF/XMLapplication/rdf+xmlFor CONSTRUCT queries
JSON-LDapplication/ld+jsonFor CONSTRUCT queries
SSEapplication/sparql-query+sseA special format for special purposes

Appendix C: Words and What They Mean

Some words in this manual might be new to you. Here's what they mean:

Word What It Means
AccountYou! Well, your identity in the system.
RepositoryA place where data lives. Like a cupboard for information.
ViewA saved question (written in SPARQL).
QuadA single fact with four parts: what, about what, the value, and where.
SPARQLThe language for asking questions about RDF data. Pronounced "sparkle."
CollaboratorSomeone you've let into your repository.
TokenA special password for computer programs to use.
PaneA section of the screen with its own tabs.
ImportBringing data in.
ExportTaking data out.
