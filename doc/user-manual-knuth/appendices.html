Appendix A: Serialization Formats

This appendix enumerates the data serialization formats supported by the system for import, export, and query result operations.

A.1 Import Formats

The following formats are accepted for data import operations:

Format Name File Extension(s) Description
Turtle.ttlTerse RDF Triple Language, a compact human-readable RDF syntax
RDF/XML.rdf, .xmlXML-based RDF serialization, W3C original standard
N-Triples.ntLine-based triple format, one statement per line
N-Quads.nqN-Triples extended with graph context
TriG.trigTurtle extended with named graph support
JSON-LD.jsonld, .jsonJSON-based linked data format
CSV.csvComma-separated values with RDF transformation

A.2 Export Formats

The following formats are available for data export:

Format Name File Extension Description
Turtle.ttlCompact RDF serialization
N-Triples.ntLine-based triple format
N-Quads.nqLine-based quad format
RDF/XML.rdfXML-based serialization
JSON-LD.jsonldJSON-based linked data
TriG.trigTurtle with named graphs
CSV.csvTabular representation

A.3 Query Result Formats

SPARQL query results may be requested in these formats:

Format Media Type Application
JSONapplication/sparql-results+jsonDefault format for SELECT/ASK queries
XMLapplication/sparql-results+xmlW3C standard XML result format
SVGimage/vnd.dydra.SPARQL-RESULTS+GRAPHVIZ+SVG+XMLGraph visualization rendering
HTMLtext/htmlHuman-readable tabular display
CSVtext/csvComma-separated values
TSVtext/tab-separated-valuesTab-separated values
Turtletext/turtleCONSTRUCT/DESCRIBE results
N-Triplesapplication/n-triplesCONSTRUCT/DESCRIBE results
RDF/XMLapplication/rdf+xmlCONSTRUCT/DESCRIBE results
JSON-LDapplication/ld+jsonCONSTRUCT/DESCRIBE results
SSEapplication/sparql-query+sseS-expression format for query algebra

Appendix C: Terminology

Definitions of technical terms used throughout this manual:

Term Definition
AccountA principal entity capable of authentication; the identity associated with a user
RepositoryA named, persistent container for RDF quads
ViewA named, persistent SPARQL query associated with a repository
QuadAn RDF statement comprising subject, predicate, object, and graph components
TripleAn RDF statement comprising subject, predicate, and object (quad without explicit graph)
SPARQLSPARQL Protocol and RDF Query Language; the standard query language for RDF
CollaboratorAn account granted access permissions to another account's repository
TokenAn authentication credential suitable for programmatic API access
PaneA tabbed interface element displaying a distinct view or editor
ImportThe operation of ingesting external data into a repository
ExportThe operation of extracting data from a repository to an external file
Namespace PrefixA short alias for a URI namespace, enabling compact notation in SPARQL
