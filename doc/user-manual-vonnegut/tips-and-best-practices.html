Words of Wisdom

Here are some things people have learned the hard way. You might as well benefit from their suffering.

Getting Around

  • The Location Bar: You can type URLs right into it. Faster than clicking.
  • Back and Forward: The browser buttons work. Use them.
  • Tabs: Open multiple things at once. Computers are good at this.
  • Keyboard Shortcuts:
    • Ctrl+Enter (Cmd on Mac): Run the query
    • Enter: Submit forms
    • Escape: Never mind, close this

The Query Editor

  • Colors: The syntax highlighting helps. Trust the colors.
  • Autocomplete: Let the computer help you type.
  • Tabs: Run multiple queries. Compare results. Learn things.
  • Formats: Try different ones. JSON for machines. HTML for eyes.
  • The Log: It tells you what happened. Read it when things go wrong.

Data

  • Backups: Export before you do anything dramatic. You'll sleep better.
  • Test First: Small imports before big ones. See what happens.
  • Async Imports: Big files? Let them run in the background. Go get coffee.
  • Watch the Progress: Those indicators mean something. Pay attention.
  • Verify: After import, run a query. Make sure it worked.

Collaborators

  • Start Small: Give read access first. Write access is a privilege.
  • Write It Down: Remember why you gave access. You'll forget otherwise.
  • Check Sometimes: Review your collaborator list. Remove the ones who left.
  • Talk to People: Tell collaborators what you expect. Assumptions cause trouble.

Performance

  • Close Tabs: Too many tabs slow things down. The computer gets tired.
  • Use Filters: Don't scroll through everything. Search for what you need.
  • Write Good Queries: Bad SPARQL is slow SPARQL. Learn to write efficiently.

When Things Break

Import Problems

Data won't go in? Check these:

  • Is the format supported?
  • Is the file too big?
  • Do you have write permission?
  • Try async for big files

Query Problems

Query won't run? Consider:

  • Is your SPARQL correct?
  • Is there data to query?
  • What does the error say?
  • Try a simpler query first

Login Problems

Can't get in? Try:

  • Check your credentials
  • Is the host URL right?
  • Clear your browser cache
  • Ask the administrator

Collaboration Problems

Sharing not working? Check:

  • Are the account names right?
  • Can you manage collaborators?
  • Did you click Save?
  • Try refreshing

The Most Important Thing

Click Save. Whatever you do, click Save. The computer doesn't know what you want until you tell it, and you tell it by clicking Save.

That's the most important tip. Everything else is details.
