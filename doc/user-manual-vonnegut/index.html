Welcome, Earthling

Listen: This is a manual about a computer program called Dydra Studio. The program lets you store data and ask it questions. The data is stored as something called RDF, which stands for Resource Description Framework. Don't worry about that. Most people don't.

Here's the thing about data: we make a lot of it. We make so much data that we need special programs to keep track of it all. This is one of those programs. It's pretty good at what it does.

What Can This Thing Do?

Dydra Studio is what programmers call a "single-page web application." That means it all happens in one browser window. No refreshing. No waiting. Well, some waiting. There's always some waiting.

Here's what you can do with it:

  • Store data in repositories (boxes for your data)
  • Ask questions using SPARQL (a language for talking to data)
  • Import and export your data (get it in, get it out)
  • Create views (saved questions you can ask again)
  • Edit information about yourself and your data
  • Share with others (if you're feeling generous)

Getting Started

The Address

Type this into your browser:

URL: https://dydra.com/ui/user

That's it. That's the door. Walk through it.

Your Browser

You'll need a modern browser. Here's what that means:

  • JavaScript needs to work (it probably does)
  • Chrome, Firefox, Safari, or Edge will all do the job
  • Get the latest version if you can

If your browser is from the last decade, you should be fine. If it's older than that, well, I admire your commitment to the past.

Two Interfaces

There are two versions of this thing:

  • User App: This is what you'll use. It's for regular humans.
  • Console: This is for administrators. It's covered somewhere else.

This manual is about the User App. That's probably what you want.

Quick Start

Here's what you do:

  1. Go to https://dydra.com/ui/user
  2. Log in (see the Login section)
  3. Do whatever you came here to do

Simple enough. Everything is simple when you break it down into steps.
